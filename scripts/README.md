# Automation Scripts

This directory contains automation scripts for the VoBee AI Assistant project.

## Available Scripts

### fetch_recent_commits.sh

Fetches all commits from the last 2 hours across all repositories accessible to the authenticated user.

**Quick Start:**

```bash
# Authenticate with GitHub CLI (recommended)
gh auth login

# Run the script
./scripts/fetch_recent_commits.sh
```

For detailed documentation, see the main [README.md](../README.md#automation-scripts) file.
